在CentOs7中安装

安装条件 内核版本CentOs6注意，CentOs7可以，而且CentOS7 系统 CentOS-Extras 库中已带 Docker，可以直接安装。

我的环境就是阿里云>CentOs7
http://wiki.jikexueyuan.com/project/docker-technology-and-combat/centos.html

https://mos.meituan.com/library/26/how-to-install-docker-on-centos/

按照极客学院教程安装安装报错：

centos7 docker 安装 file /usr/lib/systemd/system/blk-availability.service from install of device-mapper-7:1.02.107-5.el7_2.1.x86_64 conflicts with file from package lvm2-7:2.02.105-14.el7.x86_64

解决方法：
[root@localhost ~]# yum install lvm2 -y
[root@localhost ~]# yum install docker-io -y

http://www.roddypy.com/index.php/2016/03/11/centos7-yum-%E5%AE%89%E8%A3%85docker%E6%8A%A5%E9%94%99/
